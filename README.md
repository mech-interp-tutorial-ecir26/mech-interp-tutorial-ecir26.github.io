# ECIR 2026 Tutorial Site

Static one-page website for a conference tutorial, built with plain HTML and CSS.

## Files

- `index.html`: page structure and section content
- `styles.css`: Delft-inspired theme and responsive styling

## Edit content

Update these sections in `index.html`:

- Hero title, event metadata, and button links (`Paper`, `Slides`, `Code`)
- `About This Tutorial` abstract paragraph
- Organizer names and affiliations
- Schedule table rows
- Footer contact line

## Preview locally

Open `index.html` directly in your browser, or serve the folder with any static server.

## Deploy on GitHub Pages

1. Create a GitHub repository and push this folder contents to the default branch.
2. In GitHub, go to `Settings` -> `Pages`.
3. Under **Build and deployment**, choose:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (or your default branch)
   - **Folder**: `/ (root)`
4. Save, then wait for deployment. GitHub will show the site URL in Pages settings.

For project pages, your URL usually looks like:

- `https://<username>.github.io/<repository>/`
